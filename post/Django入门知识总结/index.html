    使用虚拟环境 Virtualenv

    已建在C:/users/joy/django/venv下，在pycharm中打开setting添加本地python环境

    选择刚刚创建的虚拟环境

    开启关闭virtualenv

    进入venv目录

    激活 ：cmd>D:\virtualenv\venv>Scripts\activate

    关闭 ：(venv) D:\virtualenv\venv>deactivate

    注意

    virtualenv 创建的虚拟环境与主机的 Python 环境完全无关，

    你主机配置的库不能在 virtualenv 中直接使用。

    你需要在虚拟环境中利用 pip install 再次安装配置后才能使用。

    windows下使用pycharm配置python的virtualenv环境
